# Climatelag: Technical Definition

    ## Formal Definition
    Temporal delay between greenhouse gas emissions and resulting climate impacts creating intertemporal externalities.

    ## Conceptual Background
    Climatelag is used here as an operational concept rather than a rhetorical label. It addresses intertemporal delay between emissions and realized climate damages. The mechanism focus is carbon stock persistence and slow system response creating delayed but committed impacts, which can be represented with explicit channels: emission_flow, atmospheric_stock, response_delay, damage_realization_lag, policy_delay, discounting_bias, adaptation_capacity. This repository couples concept definition with executable implementation so assumptions remain inspectable. In practice, the framework supports comparative statics, scenario simulation, and policy diagnostics. Typical applications include climate policy timing analysis, intergenerational welfare modeling, and adaptation planning prioritization. Because the index logic is transparent, researchers can vary weights and test robustness across model specifications.
